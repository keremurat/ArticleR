ArticleR - Gizlilik Politikası

Son Güncelleme: 4 Şubat 2025

Kısa Özet: ArticleR, gizliliğinize önem verir. Tüm verileriniz cihazınızda yerel olarak saklanır. Kişisel bilgilerinizi toplamıyor, paylaşmıyor veya satmıyoruz.

1. Veri Toplama

1.1 Topladığımız Veriler

ArticleR extension'ı yalnızca aşağıdaki verileri toplar:

  • Kaydettiğiniz kelimeler: Çevirilerini kaydettiğiniz kelimeler ve çevirileri
  • Ayarlarınız: Hedef dil, hover gecikmesi, otomatik okuma tercihleri
  • Sayfa URL'leri: Kelime kaydettiğinizde bulunduğunuz sayfanın adresi (opsiyonel)

1.2 Toplamadığımız Veriler

  • Kişisel bilgiler (ad, e-posta, telefon)
  • Tarama geçmişi
  • Okuduğunuz içerik
  • Lokasyon bilgisi
  • Cihaz bilgileri

2. Veri Saklama

Tüm veriler yerel olarak saklanır:

  • Kayıtlı kelimeler ve ayarlar tarayıcınızın yerel depolama alanında (Chrome Storage) tutulur
  • Hiçbir veri uzak sunuculara gönderilmez
  • Verileriniz sadece sizin cihazınızda bulunur

3. Veri Kullanımı

Toplanan veriler yalnızca şu amaçlarla kullanılır:

  • Kelime defteri: Kaydettiğiniz kelimeleri size göstermek
  • İstatistikler: Öğrenme ilerlemenizi hesaplamak
  • Tercihler: Extension'ı kişiselleştirmek

4. Üçüncü Taraf Hizmetler

4.1 Çeviri API'si

ArticleR, kelime çevirisi için MyMemory Translation API kullanır:

  • Sadece çevrilecek kelime API'ye gönderilir
  • Kişisel bilgiler gönderilmez
  • MyMemory gizlilik politikası: https://mymemory.translated.net

4.2 Sesli Okuma

Sesli okuma özelliği Chrome'un yerleşik TTS (Text-to-Speech) API'sini kullanır:

  • Ses verileri cihazınızda işlenir
  • Hiçbir ses kaydı saklanmaz veya gönderilmez

5. Veri Paylaşımı

Verilerinizi asla paylaşmıyoruz:

  • ❌ Üçüncü taraflara satmıyoruz
  • ❌ Reklam şirketleriyle paylaşmıyoruz
  • ❌ Analitik servislere göndermiyoruz
  • ❌ Veri brokerlarına vermiyoruz

6. Veri Güvenliği

  • Veriler Chrome'un güvenli storage API'sinde saklanır
  • Extension izinleri minimal seviyede tutulmuştur
  • Kod açık kaynak olup GitHub'da incelenebilir

7. Kullanıcı Hakları

Verileriniz üzerinde tam kontrole sahipsiniz:

  • Görüntüleme: Tüm kayıtlı kelimelerinizi extension popup'ından görebilirsiniz
  • Dışa Aktarma: İstatistikler sekmesinden JSON formatında indirebilirsiniz
  • Silme: İstediğiniz zaman kelimeleri tek tek veya toplu silebilirsiniz
  • Temizleme: Extension'ı kaldırdığınızda tüm veriler otomatik silinir

8. Çocukların Gizliliği

ArticleR, 13 yaş altı çocuklardan bilerek veri toplamaz. Extension herkes tarafından kullanılabilir ancak ebeveyn izni önerilir.

9. İzinler

Extension şu izinleri kullanır:

  • storage: Kelimeler ve ayarları saklamak için
  • activeTab: Aktif sekmede çeviri yapmak için
  • scripting: Web sayfalarına tooltip eklemek için
  • host_permissions (all_urls): Tüm web sitelerinde çalışabilmek için

10. Değişiklikler

Bu gizlilik politikası güncellenebilir. Önemli değişiklikler olduğunda extension üzerinden bildirim yapılacaktır.

11. İletişim

Gizlilik ile ilgili sorularınız için:

  • GitHub: GitHub Issues
  • E-posta: your.email@example.com

12. Yasal Uyum

Bu extension aşağıdaki düzenlemelere uygundur:

  • GDPR (Avrupa Genel Veri Koruma Yönetmeliği)
  • KVKK (Kişisel Verilerin Korunması Kanunu - Türkiye)
  • Chrome Web Store Developer Program Policies

Bu gizlilik politikası ArticleR Chrome Extension için geçerlidir. Extension'ı kullanarak bu politikayı kabul etmiş olursunuz.
